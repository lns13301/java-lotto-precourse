## 💡 기능 구현

1. 출력
    - [ ] 구매 완료

2. 입력
    - [ ] 구매할 금액
        - [ ] 숫자 타입 인지?
        - [ ] 올바른 숫자 인지?
    - [ ] 지난주 당첨 번호
        - [ ] 형식이 올바른지?
        - [ ] 보너스 볼
            - [ ] 형식이 올바른지?
            - [ ] 당첨 번호와 중복 되지 않는지?
        
3. 프로그램 매니저
    - [ ] 입력 받은 돈으로 구매할 수 있는 최대 로또 개수 반환
    - [ ] 로또 게임 결과
